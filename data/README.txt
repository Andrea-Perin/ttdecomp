EXPLANATION FOR THE TITLES OF THE DATA FILES

land_122_240.csv -> landscape image of resolution (122,240,3) since the color channel is RGB

land_122_240_hosvd_10_20_3.csv -> HOSVD reconstruction using ranks (10,20,3)

land_122_240_hooi_10_20_3.csv  -> HOOI  reconstruction using ranks (10,20,3)

mnist_100.csv -> dataset with 100 digits with resolution (28,28) i.e. tensor of size (100,28,28)

mnist_100_hosvd_30_10_10.csv -> HOSVD reconstruction using ranks (30,10,10)

mnist_100_hooi_30_10_10.csv  -> HOOI  reconstruction using ranks (30,10,10)
